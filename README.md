## Features

- User authentication
- Product search & category filters
- Shopping cart & checkout
- Admin dashboard
- Product & order management
- Fully responsive design

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Database:** PostgreSQL
- **Authentication:** JWT, bcrypt
